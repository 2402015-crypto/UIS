# UIS

Proyecto Expo Router con inicio en [app/login.tsx](app/login.tsx) y pantalla principal en [app/home.tsx](app/home.tsx).

## Arranque

```bash
npm install
npx expo start
```

## Estructura activa

- [app/_layout.tsx](app/_layout.tsx) define la navegación principal.
- [screens/LoginScreen.js](screens/LoginScreen.js) contiene el formulario de acceso.
- [context/AuthContext.js](context/AuthContext.js) mantiene la sesión en memoria.
